# [#](https://funtl.com/zh/gitlab/Git-的基本工作流程.html#git-的一般工作流程)Git 的一般工作流程

- 克隆 Git 资源作为工作目录。
- 在克隆的资源上添加或修改文件。
- 如果其他人修改了，你可以更新资源。
- 在提交前查看修改。
- 提交修改。
- 在修改完成后，如果发现错误，可以撤回提交并再次修改并提交。
